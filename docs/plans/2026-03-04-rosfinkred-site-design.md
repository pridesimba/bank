# РОСФИНКРЕД — Дизайн сайта кредитного продукта

## Обзор
Одностраничный лендинг для оформления кредита (рефинансирования). Многошаговая анкета с отправкой заявки в Telegram. Хостинг на Netlify.

## Стек
- HTML + CSS + Vanilla JS
- Netlify Functions (серверная отправка в Telegram)
- Деплой через git push на Netlify

### 3. Преимущества (4 карточки)

- От 250 000 рублей

### 4. Как это работает (4 шага)

### 5. Калькулятор платежей
- Слайдер суммы кредита (250 000 — 3 000 000 ₽)
- Слайдер срока (1 — 15 лет)
- Формула аннуитетного платежа: P = S * (r*(1+r)^n) / ((1+r)^n - 1)
- Ставка по умолчанию: 5.5% годовых
- Мгновенный пересчёт при движении слайдеров

### 6. Многошаговая анкета (7 шагов)

**Шаг 1**: Сумма для погашения кредитов
- 3 кнопки: 250-500к / 500к-1кк / 1-3кк (выбор одного)

**Шаг 2**: Комфортный ежемесячный платёж
- Поле ввода числа с подписью «Сумма платежа в месяц»

**Шаг 3**: Нужна отсрочка ближайшего платежа?
- Да / Нет

**Шаг 4**: Есть автокредит?
- Да → «Мы не работаем с автокредитами», стоп-экран
- Нет → следующий шаг

**Шаг 5**: Есть имущество зарегистрированное на вас?
- Нет → следующий шаг
- Да → «Сколько?» (1 / 1-3 / Более 3) → следующий шаг

**Шаг 6**: Подтверждение региона (СПб / ЛО)
- Согласен → следующий шаг
- Не согласен, я из другого региона → стоп-экран

**Шаг 7**: Финальный шаг
- Вероятность одобрения: рандом 80-97%
- Выпадающий список: способ связи (Telegram / WhatsApp / Звонок сотрудника РОСФИНКРЕД)
- Поле: Номер телефона (+7(***) ***-**-**)
- Поле: ФИО
- Если выбран Telegram: поле «Username в Telegram (при наличии)»

**После отправки**:

## Telegram-интеграция
- Токен бота хранится в .env (не в коде!)
- Netlify Function принимает POST с данными формы
- Формирует сообщение с эмодзи и отправляет в чат
- Chat ID настраивается в .env

## Дизайн
- Цвета: синий (#1a56db) + белый + акцент
- Шрифт: Inter (Google Fonts)
- Полностью адаптивный (мобилка/планшет/десктоп)
- Плавные анимации переходов в форме
